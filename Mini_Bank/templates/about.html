About This Project

Mini Bank is a web-based banking application designed to emulate the core features of a real-world banking system. Developed as a learning project using Django and Bootstrap, it offers a seamless experience for users, employees, and administrators alike.

  • ✔️ Users can submit account creation requests and receive updates via email.
  • ✔️ Managers can approve or reject account requests through a secured interface.
  • ✔️ Approved users can log in using account numbers and perform fund transfers.
  • ✔️ Employees can credit or debit funds into any customer account as required.
  • ✔️ Every critical action is logged and notified via real-time email updates.

This project demonstrates practical applications of authentication, role-based access control, form handling, database operations, and email integration. It serves as an ideal example of a beginner-to-intermediate level full-stack development project.
